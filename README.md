## Технологии

* Vite (сборщик)
* Bootstrap 5 (стилизация)
* JavaScript (промисы)

## Запуск локально

1. Клонируйте репозиторий.
2. Выполните `npm install`.
3. Запустите `npm run dev`.

## Структура проекта

* `index.html` — главная страница.
* `src/` — исходный JS‑код.
* `styles/` — стили.
* `package.json` — зависимости и скрипты.
